Navegar al directorio donde esta el dockerfile:
	cd /_data/consolidainfo


Construye la imagen del Docker:
Al realizar cualquier cambio en el codigo, es necesario volver a ejecutar este comando
Ejecuta el siguiente comando para construir la imagen del Docker.

	docker build -t consolidainfoarchivos_apache .


Ejecuta el contenedor:
Despu√©s de construir la imagen, puedes ejecutar un contenedor basado en esa imagen.

	docker run --name pr_consolidainfoarchivos -p 15107:3080 consolidainfoarchivos_apache


Detener y eliminar el contenedor:
Cuando hayas terminado de probar, puedes detener el contenedor con el siguiente comando:

	docker stop pr_consolidainfoarchivos

Y para eliminar el contenedor:

	docker rm pr_consolidainfoarchivos



Para reiniciar:

Detener el contenedor:
	docker stop pr_consolidainfoarchivos

Iniciar el contenedor
	docker start pr_consolidainfoarchivos
